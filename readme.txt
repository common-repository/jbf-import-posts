== Description ==

Tested with **Wordpress 3.1**

Import latest posts from John Benson Fitness Host Blog to your Blog via RSS feed. WP-Cron settings for automatical import of posts in a regular interval.

Plugin Option Page to set the post category and count of posts to import.


== Support ==

Please post your question in the forum under **[Support forum](http://wordpress.org/tags/jbf-import-posts?forum_id=10#postform)** .  


== Installation ==

1.  extract plugin zip file and load up to your wp-content/plugin directory
2.  Activate Plugin in the Admin => Plugins Menu
